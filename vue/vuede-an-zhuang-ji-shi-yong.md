#### 安装

```bash
npm install vue
```

#### 使用脚手架构建空项目

```
npm install -g @vue/cli   //安装vue-cli3.0 版本
vue create myApp          //创建项目

使用vue-cli3.0版本安装的项目会将webpack配置文件隐藏，如果想要修改webpack中的配置，需在根目录下手动创建vue.config.js文件，
具体配置见文档[https://cli.vuejs.org/zh/guide/webpack.html#%E7%AE%80%E5%8D%95%E7%9A%84%E9%85%8D%E7%BD%AE%E6%96%B9%E5%BC%8F]

npm install 

```
