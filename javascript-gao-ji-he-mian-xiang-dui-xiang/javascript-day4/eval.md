### eval

> eval\(\) 函数可计算某个字符串，将字符串转换成JavaScript代码，执行其中的JavaScript代码

```js
eval('1 + 2')    // 3
```

注意: 解析JSON格式字符串，将{}解析为代码段

解决方法:

1. JSON格式的字符串 前面拼接 "var 变量名=
2. eval\("\("+JSON格式的字符串+"\)"\);

### Function和eval的区别

* 共同点：都是将字符串转换成js代码．

* 不同点：Function创建的是函数，不会直接调用，只有手动调用的时候才会执行；eval把字符串转换成js代码后直接就执行了
